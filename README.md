# streamlit-uni-demo



## Getting started

[Windows] Open Anaconda prompt in Admin mode.  

```
conda create --name uni-demo-streamlit python==3.10
conda activate uni-demo-streamlit
cd path ...navigate to path...
conda install -c conda-forge streamlit
streamlit hello
```
